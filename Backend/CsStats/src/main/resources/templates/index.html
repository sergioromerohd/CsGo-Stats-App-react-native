Get started

    API Endpoint

            http://192.168.1.102:8080

Cs2 Stats is the application that will allow you to review, compare, and analyze the competitive match statistics of all CS players. You can also advance in the rankings and explore inventories.

This API is free to use, by the moment it only being accesible from inside the app, more comming soon.

Post Users


# Here is a curl example
curl \
-X POST http://192.168.1.102:8080/user \
-H 'Content-Type: application/json'
-d '{
    additionalParameters: '',
    avatarUrl: '',
    platformSlug: '',
    platformUserHandle: '',
    platformUserId: 1234,
    platformUserIdentifier: 12354,
};'

To post users you need to make a POST call to the following url :
http://192.168.1.102:8080/user
The body of the request must be a JSON object with the following fields : 



Result example :

{
  query:{
    additionalParameters: '',
    avatarUrl: '',
    platformSlug: '',
    platformUserHandle: '',
    platformUserId: 1234,
    platformUserIdentifier: 12354,
}
  result: 'Done' | 'Error' | 'Dupe'
}

QUERY PARAMETERS

Field Type Description
additionalParameters String Extra info, only use for backend, not saving
avatarUrl String Avatar URL for steam logo
platformSlug String Plataform sllugh from steam, only use for backend, not saving.
platformUserHandle String User name, used to be something human understranding
platformUserId String User id, steam id64
platformUserIdentifier String If steam, its the same as platformUserId

Errors

The Westeros API uses the following error codes:

Error Code Meaning
X000 Some parameters are missing. This error appears when you don't pass every mandatory parameters.
X001 Unknown or unvalid secret_key. This error appears if you use an unknow API key or if your API key expired.
X002 Unvalid secret_key for this domain. This error appears if you use an API key non specified for your domain. Developper or Universal API keys doesn't have domain checker.
X003 Unknown or unvalid user token. This error appears if you use an unknow user token or if the user token expired.
